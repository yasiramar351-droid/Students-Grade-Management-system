# 🎓 Students Grade Management System

A C-based application designed to calculate student mark averages, classify academic grades, and query student records from an external CSV database.

## 📁 Repository Contents
* `main.c`: Source code written in C language.
* `Estudents grades.csv`: Sample database file containing student records.
* `Presentation.pdf`: Project presentation slide deck.

## 🚀 Features
1. **Interactive Grade Calculator:** Accepts marks for 7 core subjects, calculates average percentage, and assigns letter grades (A, B, C, D, F).
2. **CSV Database Search:** Parses an external CSV file using `strtok` and `strcmp` to search and display individual student records.
